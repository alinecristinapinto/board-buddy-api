
# BoardBuddy API

BoardBuddy é uma plataforma visa conectar uma vasta comunidade de entusiastas de jogos, permitindo que compartilhem e peguem emprestado uma ampla variedade de jogos. [TODO]

## Instalação

```bash
  npm install 
```

```bash
  npm start 
```
